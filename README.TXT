## INSTRUÇÕES PARA OBTER O EXECUTAVEL DO PROGRAMA ##
- pelo terminal, ir até a pasta /src e executar os comandos:
    javac modelo/*.java
    javac controle/*.java
    javac visao/*.java
    
- para rodar o programa, chame o executável tendo como argumento o endereço do arquivo de entrada
    java visao.Visao ../entradas/simples.txt
